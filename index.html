Tipos de Comercio Electronico

B2B: este tipo de transacciones se producen entre dos empresas. B2C: Es el que se da entre una empresa y un consumidor. C2B: Este tipo de comercio se da entre una persona que ofrece su servicio o productos a un empresa. C2C: Este comercio se da entre consumidores y usuarios finales, sin que intervengas ninugna empresa.

¿Qué es un sitio WEB?

Un sitio Web es un conjunto de paginas web bien organizadas. Las paginas Web de las que conforman el sitio deben estar desarrolladas bajo un codigo HTML y estar alojadas a un dominio que en palabrs simples, sera el lugar que hará el sitio Web. y puede visualizar en cualquier tipo de navegador

Aspectos a considerar para la creación de un sitio WEB

Homepage llamativa para que los cliente se lleven una buena impresion de la empresa, rapider de carga,sitio confiable, seguro y protegido,imagen de marca y buen diseño botones de las redes sociales y un navegador.

Herramientas y paginas que nos facilitan poder crear nuestra pagina WEB

En clases pude obtener el conocimiento de herramientas y paginas que nos puede facilitar la creacion de una pagina WEB, por ejemplo utilizar startbootstrap, github, Webnode, etc.
